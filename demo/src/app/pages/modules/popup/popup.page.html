Popup

A popup displays additional information on top of a page

Examples

Popup

An element can specify popup content to appear

Template

You can use a template element for a popup.

If you need to access a local variable, such as within an *ngFor, either place the template within the scope of the repeated elements, or bind popupTemplateContext to the variable and access it in the template as popup.context.

Adding the autofocus attribute to an element inside the template will give it focus when the popup opens.

Placement

You can control the placement (position) of each popup.

Size

A popup can vary in size.

Width

A popup can be extra wide to allow for longer content

A flowing popup have no maximum width and continue to flow to fit its content
