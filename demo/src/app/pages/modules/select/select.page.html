Select

A select allows a user to select one or more values from a series of values

Examples

Selection

A select can be used to select between choices in a form

Full keyboard navigation is supported.
Important Note

The select component is unusual in the fact that it requires you to manually provide the options within. This is so that you can customise the menu, adding headers and dividers etc. as in Semantic UI.

However, it means that in order to always display the correct options, you must bind your *ngFor to the filteredOptions property on the select. This is shown in each of the examples.

Remote Lookup

A select can allow a user to search through a list of choices from an API


Remote lookups are facilitied by a lookup function of type LookupFn<T, U>.

T is the type of each option, and U is the type of the property specified by valueField (default is T if no valueField specified).

For multi selects, the initial argument is of type U[], and the lookup function always returns a Promise<T>.
